# Audio Files — Get Underway Maritime English L1

Place MP3 recordings in this folder using the slug naming convention:

| English Term / Phrase       | File name              |
|-----------------------------|------------------------|
| Bow                         | bow.mp3                |
| Stern                       | stern.mp3              |
| Port                        | port.mp3               |
| Starboard                   | starboard.mp3          |
| Slow ahead                  | slow_ahead.mp3         |
| Stop engines                | stop_engines.mp3       |
| Midships                    | midships.mp3           |
| Let go!                     | let_go.mp3             |
| Anchor is holding.          | anchor_is_holding.mp3  |
| Understood.                 | understood.mp3         |

## Naming Rules
- All lowercase
- Spaces → underscores
- Remove punctuation (apostrophes, dots, exclamation marks etc.)
- `(s)` → `s`   e.g. `Stop engine(s)` → `stop_engines.mp3`

If an MP3 file is not found, the system automatically falls back to
browser Text-to-Speech (Web Speech API) so the object remains fully
functional at all times.
